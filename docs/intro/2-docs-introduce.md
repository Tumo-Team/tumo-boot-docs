## 下载文档

1. `clone`项目：`git clone https://github.com/Tumo-Team/tumo-boot-docs.git`
2. `install`项目：`yarn install`



![image-20210808185742204](http://cdn.tycoding.cn/20210808185742.png)

## 运行文档

使用`yarn dev`或使用`npm run dev`命令运行项目。
