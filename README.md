**特点：**

1. 代码简洁、规范。
2. 提供最后端系统最基础的权限体系模块。
3. 项目文档将涉及前后端、运维，带你一步步学习怎么优雅的开发。
4. 基于SpringBoot最新版，单体架构，前后端分离。前端采用Ant-Design-Vue。
5. 提供最最最基础、完善的项目开发文档，所有源码及文档都开源在GitHub上。

## 本地运行

```bash
# 克隆本仓库
$ git clone https://github.com/Tumo-Team/tumo-boot-docs.git

# 或者使用 npm
$ yarn install

# 启动开发服务器
$ yarn dev
```
